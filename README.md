1️⃣ What is the difference between var, let, and const?
Ans: var is an old JavaScript variable. It's value can be changed and can declare again. Not used much now. let is a variable that can be changed later. But const variable value cannot be changed after initialized.
  
2️⃣ What is the spread operator (...)?
Ans: Spread operator expands elements of an array or object.

Example:
const arr1 = [1,2]
const arr2 = [...arr1,3]

Result: [1,2,3]

3️⃣ What is the difference between map(), filter(), and forEach()?
Ans: map() makes a new array by changing each item.
     filter() makes a new array with items that match a condition.
     forEach() just loops through items, it does not create a new array.

4️⃣ What is an arrow function?
Ans: arrow functions are shorter syntax for writing functions.

Example: const sum = (a,b) => a+b

5️⃣ What are template literals?
Ans: Template literals let put variables inside a string using backticks.

Example:
const name = "World"
console.log(`Hello ${name}`)

Result: Hello World
